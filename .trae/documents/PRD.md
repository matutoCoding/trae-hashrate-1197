## 1. 产品概述

脚手架周转租赁管理系统，面向建筑租赁企业，解决脚手架租赁业务中分时段阶梯计费、租赁排期管理、超时自动释放及候补队列补位等核心痛点。系统通过自动化计费规则配置实现高峰/平峰/低谷多档费率智能拆分计算，结合排期系统自动管理脚手架资源调度，提升租赁运营效率与资源利用率。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 系统管理员 | 后台创建 | 全部功能：计费规则配置、费率维护、脚手架管理、账单查询、候补管理、数据统计 |
| 运营人员 | 后台创建 | 脚手架建档、排期管理、候补登记、账单查询、库存盘点 |
| 财务人员 | 后台创建 | 费率配置、账单生成与导出、费用核算 |

### 2.2 功能模块

1. **分时段计费模块**：计费规则配置、时段费率表维护、跨档时长拆分计算
2. **账单生成模块**：账单明细、费用汇总、账单导出
3. **脚手架排期模块**：脚手架建档、排期日历、超时自动释放、杆件数量盘点
4. **候补补位模块**：候补排队登记、自动补位通知、候补状态追踪

## 3. 核心流程

### 3.1 租赁计费流程

客户提交租赁申请 → 选择脚手架类型与数量 → 选择租用起止时间 → 系统自动拆分跨费率时段 → 按各时段费率分别计算 → 分段费用汇总 → 生成账单明细 → 确认租赁 → 创建排期

```mermaid
flowchart LR
    A["客户提交租赁申请"] --> B["选择脚手架与数量"]
    B --> C["选择租用起止时间"]
    C --> D["系统识别费率切换点"]
    D --> E["拆分各时段时长"]
    E --> F["按对应费率计算"]
    F --> G["分段费用汇总"]
    G --> H["生成账单明细"]
    H --> I["确认并创建排期"]
```

### 3.2 超时释放与候补补位流程

租赁到期 → 检测是否续租 → 未续租超时 → 自动释放脚手架资源 → 查询候补队列 → 按优先级通知首候补 → 候补确认 → 分配资源 → 更新排期与候补状态

```mermaid
flowchart LR
    A["租赁到期"] --> B{"客户是否续租"}
    B -->|"是"| C["续租处理"]
    B -->|"否，等待超时"] --> D["超时自动释放资源"]
    D --> E["查询候补队列"]
    E --> F["按优先级取首位候补"]
    F --> G["发送补位通知"]
    G --> H{"候补是否确认"}
    H -->|"是"| I["分配资源创建排期"]
    H -->|"否"| J["移除候补，取下一位"]
```

## 4. 用户界面设计

### 4.1 设计风格

- 设计定位：工业/工程管理风格，专业稳重，数据密集型界面

- **主色调**：深邃藏蓝 `#1e3a5f`，辅以钢铁灰 `#2c3e50`
- **辅色**：警示橙 `#e67e22`（高峰时段标识）、成功绿 `#27ae60`（平峰/空闲）、警戒红 `#e74c3c`（低谷/超时）、信息蓝 `#3498db`（候补）
- **按钮风格**：直角硬朗工业风，2px描边，hover微凸起效果
- **字体**：标题使用"Oswald"（工业感强的无衬线字体），正文使用"Roboto Mono"等宽字体（数据清晰易读）
- **布局风格**：卡片式分区，网格布局，顶部固定导航+左侧功能菜单
- **图标风格**：线性工业风图标，与建筑工程主题呼应

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 仪表盘 | 数据概览 | 大型数据卡片、趋势折线图、环形进度图、待办事项列表、过期提醒高亮 |
| 计费规则配置 | 分时段计费 | 表单布局、时段滑块选择器、费率输入框、时间轴可视化 |

### 4.3 响应式设计

桌面端优先，1440px为基准设计，12列栅格系统。侧边菜单在平板端折叠为汉堡菜单，移动端单列堆叠，表格支持横向滚动。

### 4.4 动效设计

- 仪表盘数据卡片加载时数字滚动动画
- 排期日历色块悬停放大阴影
- 候补通知弹出滑入动画
- 费率切换点时间轴流动光效
- 页面切换淡入过渡（0.3s ease）
